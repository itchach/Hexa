📘 Logic Cheat Sheet

∧
AND (Conjunction)
True only when both are True
T ∧ T = T T ∧ F = F F ∧ T = F F ∧ F = F
∨
OR (Disjunction)
True when at least one is True
T ∨ T = T T ∨ F = T F ∨ T = T F ∨ F = F
¬
NOT (Negation)
Inverts the truth value
¬T = F ¬F = T
→
IMPLIES (Conditional)
False only when True → False
T → T = T T → F = F F → T = T F → F = T
↔
BICONDITIONAL (Iff)
True when both values are equal
T ↔ T = T T ↔ F = F F ↔ T = F F ↔ F = T
Operator Precedence (high → low):
¬ (NOT)  ›  ∧ (AND)  ›  ∨ (OR)  ›  → (IMP)  ›  ↔ (BICON)
